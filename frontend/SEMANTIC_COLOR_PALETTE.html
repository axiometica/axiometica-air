7. CSS Variables Template (For Global Settings)

Add to tailwind.config.js or create theme.css
/* Base Colors */
--color-bg-base: #0f1419; /* Page background */
--color-bg-surface: #1a1f2e; /* Modal/Overlay */
--color-bg-card: #2d3748; /* Cards (CONFIRMED) */
--color-border: #3d4557; /* Borders */
--color-text-primary: #e8eef5; /* Main text */
--color-text-secondary: #a0aec0; /* Labels */
--color-text-tertiary: #7a8ba3; /* Helper text */
/* Semantic: Success */
--color-success-border: #15803d;
--color-success-text: #86efac;
--color-success-hover: #10b981;
--color-success-bg: #064e3b;
/* Semantic: Critical */
--color-critical-border: #dc2626;
--color-critical-text: #fca5a5;
--color-critical-hover: #991b1b;
--color-critical-bg: #7f1d1d;
/* ... and so on for other colors ... */
